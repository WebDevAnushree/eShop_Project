
# 🛍️ eShop - React E-Commerce Application

A modern and responsive **E-commerce Web Application** built using **React.js**.
Users can browse products, add to cart, update quantity, and view checkout summary.

---

## 🚀 Features

* 🛒 Add to Cart functionality
* ➕ Increase / Decrease Quantity
* ❌ Remove Items from Cart
* 💰 Checkout Price Calculation
* 📱 Fully Responsive Design
* ⭐ Product Rating UI
* 🔍 Search Products
* 🎨 Clean UI & Modern Design

---

## 🛠️ Tech Stack

* React.js
* JavaScript
* CSS3
* HTML5
* React Context API (State Management)
* DummyJSON API

---


## 🔗 API Used

DummyJSON API

```
https://dummyjson.com/products
```

---

## ⚙️ Installation

Clone the repository

```
git clone https://github.com/WebDevAnushree/eShop_Project
```

Go to project folder

```
cd eshop
```

Install dependencies

```
npm install
```

Start project

```
npm start
```
